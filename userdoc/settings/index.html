{% include screenshot.html item='i316.gif' %} Form Tools 2 comes with a great number of new features over its precessor. To help keep all the new settings categorized and organized, the settings section is split into five tabs. The following pages explain the contents of each tab, one by one.

Whither the WYSIWYG tab?

For users who are familiar with 2.0.x versions of the script, you'll notice the WYSIWYG tab has disappeared. Egads! But don't worry: the default settings for the WYSIWYG field are still around, just now found in the TinyMCE field type module. At the time of writing this, all the same functionality is still available - and now it'll be easier to expand on it.
