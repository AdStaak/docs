This module has been incorporated as part of Form Tools Core as of Form Tools 3. You will no longer need to handle it separately.

The Core Field Types module is included in all Form Tools download packages from 2.1.5 up to 3.0.0, where it has been incorporated into the core). It is used by the installation script to populate the database with the Core field types for use in your forms. Namely:

  • Textboxes
  • Passwords
  • Textareas
  • Radio buttons
  • Checkboxes
  • Dropdowns
  • Multi-select dropdowns
  • Dates
  • Times
  • Phone Numbers
  • Code/Markup fields

This is a special module. Since it handles key functionality, it cannot be uninstalled.
