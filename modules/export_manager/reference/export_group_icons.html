Export Group Icons

Assigning an icon to each export group is a nice way to help visually identify the groups. e.g. the XML and Excel icons are common enough to be immediately recognizable to most users, and let them know all they need to about the export type.

By default, there are only 4 icons .To add a new one, just FTP the icon file(s) to your /modules/export_manager/images/icons/ folder. Then the next time you load the Add / Edit export group page, your icon will appear. For maximum compatibility with all browsers, we recommend you upload them in gif or jpg format.

There are many free icon resources on the web.

  • Crystal Clear icons
  • List of open source icon sets

If you know of other good (free!) icon resource websites, just add them to this page.
