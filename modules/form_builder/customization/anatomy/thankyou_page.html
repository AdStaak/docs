Thankyou Page

Compared the last two Template Types, the Thankyou Page content is very basic. The Form Builder contains a special field to allow the Form Tools administrator to enter whatever content they want to appear on this page. So, all that needs to be done is include that value through the {% raw %}{{$thankyou_page_content}}{% endraw %} placeholder. It's often a nice idea to include the navigation as well - just for consistency and appearance. The Thankyou page finalizes and clears the form sessions so a user can't go back and edit his form submission values. As such, the navigation shouldn't permit the user to return to a particular form page.
