Resources

{% include screenshot.html item="i377.jpg" %} All generated forms contain some CSS provided by the Form Builder / Form Tools to display and render the field types properly. But in order to style each form differently, you can use Template Set Resources. What's special about Resources is that they can contain Smarty logic to output different content based on whatever criteria you want - for example, the choice selected for a placeholder. This lets you change what's outputted for each form.

Resources can be either CSS or JS, but by and large, CSS will be by far the more common usage.
