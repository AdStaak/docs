Header

The Header Template Type contains the beginning on your HTML - the opening , and opening tag.

The Form Builder generates forms with a lot of dynamic, javascript-dependant functionality, like Google Map fields, file uploads, WYSIWYG fields, field validation and so on. As such, the Header template must contain the {{$required_resources}} placeholder. That includes all the necessary javascript and CSS into the page.
