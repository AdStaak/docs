Form Builder Error Codes

In case something serious goes wrong with a Form Builder-published form, it will output a simple error code. These shouldn't ever occur, but if they do, the following description should give you a little better idea of what's going on.

Error Code Explanation
FB100 The form associated with this form builder form no longer exists.
FB101 The View associated with this form builder form no longer exists.
FB102 One of the specific Template Set templates being used by this published form no longer exists.
