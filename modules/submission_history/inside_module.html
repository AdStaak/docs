Inside the module

Alright! Let's look at the content of the module itself.

Log into Form Tools and go to the Modules page. There, browse for the Submission History module and select it. If this is the first time you're going to the module, you'll be asked to create the history tables. Do so, then proceed. If you don't have any forms added yet, you won't be able to continue.
