{% include screenshot.html item="edit_submission_multiple_files.png" %} The File Upload module handles file uploading for Form Tools. It lets you upload files through your internal, external and Form Builder forms. Up until version 2.1.0 of this module, you could upload a single file through a single form field. But now you have the option to upload as many files as you want through a form field. See the uploading multiple files page for more information.
