Validation tab

{% include screenshot.html item="i241.gif" %}

As mentioned at the top of the tab, although adding in field validation is on the roadmap to be added to the Form Tools Core, at the time this module was developed it was not available. So, this tab lets you add in some basic validation.

Just check those fields you want to be required, and add in the error string that should appear if they're not filled in.
