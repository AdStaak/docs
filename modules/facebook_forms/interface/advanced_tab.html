Advanced tab

{% include screenshot.html item="i238.gif" %}

The final "Advanced" tab contains a few remaining settings.

The first two settings relate to the Facebook "Like" button. This is a very nice feature to add to your Form Tools form. To add it, all you need to do is enabled it (set the first setting to "yes"), then provide the full URL of your App in the second field.

The final setting lets you control the width of the questions column on your Facebook form. By default, the column width is set to 180 (px).
