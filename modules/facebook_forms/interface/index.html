The module interface

{% include screenshot.html item="i243.gif" %}

The module interface is very straightforward. The main page of the module lists all of your Facebook forms where you can choose to delete or edit them. As explained in the tutorial, the "Facebook Canvas URL" column displays the URL needed by Facebook to pinpoint exactly where your "App" exists.

The edit pages are arranged in four tabs:

  • Main tab
  • Labels / Text tab
  • Validation tab
  • Advanced tab
