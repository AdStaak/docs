Example Modules

Because nothing beats the hands-on approach, here are three "Hello World" modules that you can install on your Form Tools installation to see how they work. See below for a link to instructions on how to install modules.

Hello World!
Learn More A simple module containing just the basics. Outputs a simple "Hello World!" message to the screen.
Hello Tabs!
Learn More A common technique for separating your content visually on the page is to use tabs to separate the data. This module demonstrates how to implement a tabset in your pages.
Hello Database!
Learn More This module demonstrates how to interact with the database with three simple examples:
  • The installation / uninstallation scripts are included which create a table of random numbers.
  • The numbers are viewable via a basic UI
  • A single setting is retrieved & stored in the settings database table
