This method deletes a client account. Note: it can only deletes client accounts, not the administrator. It completely removes all references to the client account from the database, including client-form and client-view omit lists, as email recipients and everywhere else.

Usage is very simple. It accepts a single parameter: the unique account ID.

{% codemirror php %} deleteClientAccount($account_id); ?> {% endcodemirror %} {% include close_section.html %}
