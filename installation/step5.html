{% include screenshot.html item='i346.gif' %} This step creates administrator user account and (behind the scenes) does all the exciting stuff, like populating the database and installing the modules. It may take a few moments to complete.

The administrator's username needs to be alphanumeric (a-Z, 0-9) only. It will prompt you if what you entered isn't permitted.
